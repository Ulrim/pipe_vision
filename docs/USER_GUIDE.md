# AIVIS 사용자 가이드 (USER_GUIDE.md)

> 작업자 HMI 와 관리자 대시보드 화면 사용법 (CLAUDE.md §5 M6/M10~M13, §12 인수 산출물).
> 설치/운영은 [`OPERATIONS.md`](./OPERATIONS.md), API 는 [`API.md`](./API.md) 참조.

## 1. 역할별 권한 (RBAC)

로그인 계정은 3개 역할 중 하나다. 위계: `admin` > `quality` > `operator`.

| 기능 | operator(작업자) | quality(품질관리자) | admin(관리자) |
|---|---|---|---|
| 검사이력/통계/이미지 조회 | O | O | O |
| NG 재확인(수동 판정 입력) | O | O | O |
| KPI 요약 조회 | O | O | O |
| 기준정보(품목) 등록/수정 | X | O | O |
| KPI 수기입력(공수/리드타임/Claim) | X | O | O |
| KPI 리포트(PDF/엑셀) 생성 | X | O | O |
| 로그 조회 | X | O | O |
| 사용자 계정 관리 | X | X | O |
| 품목 삭제 | X | X | O |

> 초기 admin 계정은 `.env` 의 `AIVIS_SEED_ADMIN_USER/PASSWORD` 로 시드된다(운영 전 비밀번호 변경).
> 추가 계정은 admin 이 발급한다.

## 2. 작업자 HMI (http://localhost:5173)

현장 대형 디스플레이/터치 최적화. 실시간 검사결과를 WebSocket(`/ws/live`) 으로 즉시 표시한다.

### 2.1 실시간 검사 화면
- **현재 검사 카드**: 마지막 검사 제품의 원본/결과 이미지, 측정 길이(mm)·기준·편차,
  최종 **OK/NG** 판정, 불량유형 배지를 표시한다.
- **OK/NG 표기**: 색(녹/적) + 아이콘/텍스트 **이중 표기**(색약 고려). 색만으로 구분하지 않는다.
- **불량유형 배지**: `LEN`(길이) / `OIL`(유분기) / `DIS`(변색) / `SCR`(스크래치) / 복합(MULTI = 2종 이상).
- **최근 피드**: 직전 검사들이 시간순으로 누적 표시되어 흐름을 한눈에 본다.
- **연결 표시기**: 서버 연결 상태(연결/끊김). WS 끊김 시 자동 재연결되며 마지막 상태를 유지한다.

### 2.2 NG 알람 대응
- NG 발생 시 화면 상단 **알람 배너**가 즉시 뜬다(표시 지연 목표 ≤200ms).
- **연속 NG**: 동일 카메라에서 연속 NG 가 임계(`AIVIS_CONSEC_NG_THRESHOLD`, 기본 3) 이상이면
  강조 알람(consecutive_ng)이 추가로 발생한다 → 설비/공정 점검 신호. 관리자 확인을 요청한다.
- 경광등/부저는 별도 신호 발신 인터페이스로 연동된다(물리 결선은 도입기업 측, 스코프 §2.2).
- 대응: 해당 제품을 분리/표시하고, 필요 시 아래 재확인을 입력한다.

### 2.3 NG 제품 재확인 (수동 확인 결과 입력)
- 검사 카드/피드에서 NG(또는 재확인 대상, `review_flag`) 제품을 선택하면 **재확인 다이얼로그**가 열린다.
- 작업자가 실물을 육안 확인 후 **수동 판정**(OK/NG)을 입력한다 → `PATCH /inspection/{id}/review`.
- 입력 시 재확인 플래그가 해제되고, 수동 판정은 **검사불량률(오검/미검)** 산출에 반영된다(§KPI).
- 권한: operator 이상 누구나 입력 가능.

## 3. 관리자 대시보드 (http://localhost:5174)

로그인 후 좌측 네비게이션으로 페이지를 이동한다. 대용량 조회는 서버 페이지네이션.

### 3.1 검사이력 (Inspections)
- 필터 조합 검색: **LOT / 품목 / 검사일자(from~to) / 최종판정(OK·NG)**.
- 결과 테이블 → 행 선택 시 상세(원본/결과 이미지, 길이값/편차, 표면 점수, 불량유형, 재확인 결과).
- 이미지/CSV 다운로드 지원.

### 3.2 통계 (Statistics)
- **불량유형별 분포**(파이) + **월별/일자별 추이**(트렌드 차트).
- 품목·기간 필터로 불량 패턴을 분석한다. 고밀도 시계열은 ECharts, 요약은 Recharts.

### 3.3 KPI (KPI)
- §1.1 산출식 기반 자동 KPI 를 게이지/카드로 표시(목표 대비 현재값).
  - 공정불량률(ppm) = NG수 ÷ 총검사수 × 1,000,000 (목표 ≤600)
  - 검사불량률(%) = (오검+미검) ÷ 총검사수 × 100 (목표 ≤30)
  - 자동검사율(%) = 판정완료 ÷ 검사대상 × 100 (목표 100)
  - 저장·MES 연계율(%) = mes_synced ÷ 전체 × 100 (목표 100)
  - 평균 처리속도(ms) (목표 ≤300ms/ea)
- **KPI 수기입력**(quality+): 작업공수 지수 / 리드타임(일) / Claim 건수를 월 단위 입력
  (자동 산출 불가 항목) → `POST /kpi/manual`.

### 3.4 기준정보 (Master, quality+)
- 품목별 **기준 길이 / 허용 공차(+,−) / 픽셀-mm 환산계수 / 표면 임계값(유분기·변색·스크래치) /
  촬영 레시피**를 등록·수정한다.
- 수정 시 version 이 증가하고 변경자/시각이 기록된다(변경 이력). 삭제는 admin 전용.
- 비전 추론은 이 기준정보를 읽어 판정한다(임계값 하드코딩 없음). 현장 보정은 여기서 한다.

### 3.5 월간 품질 리포트 (Report, quality+)
- 기간(월) 선택 → KPI/불량유형/일자별 표를 **미리보기** 후 내보내기.
- 내보내기: **PDF**(reportlab) 또는 **엑셀(XLSX)**(openpyxl) → `GET /kpi/report?period=YYYY-MM&fmt=pdf|xlsx`.
  파일은 `aivis_kpi_YYYY-MM.{ext}` 로 다운로드된다.
- 내용: §1.1 KPI, 불량유형별 집계, 일자별 검사수/불량수.

## 4. 자주 묻는 운영 시나리오

| 상황 | 화면/조치 |
|---|---|
| 방금 NG 가 떴다 | HMI 알람 배너 확인 → 제품 분리 → 재확인 다이얼로그에 수동 판정 입력 |
| 같은 라인에서 NG 연발 | HMI consecutive_ng 알람 → 설비/조명/치구 점검, 관리자 호출 |
| 특정 LOT 불량 추적 | 대시보드 Inspections 에서 LOT 필터 → 상세/이미지 확인 |
| 월말 품질보고 | 대시보드 Report 에서 해당 월 선택 → PDF/엑셀 내보내기 |
| 신규 품목 추가 | 대시보드 Master(quality+) 에서 기준길이/공차/임계값/레시피 등록 |
| 작업공수·Claim 집계 | 대시보드 KPI 수기입력(quality+) 월 단위 입력 |
| 계정 추가 | admin 이 사용자 관리에서 발급(역할 지정) |

> OK/NG 는 항상 색+아이콘 이중 표기를 신뢰한다. 색상만으로 판단하지 않는다.
